## 📌 Internship Tasks Completed

### 🔹 Task 1: Credit Scoring Model

**Objective:**
Predict an individual's creditworthiness using past financial data.

**Approach:**
Developed a machine learning classification model to evaluate whether an individual is likely to be creditworthy. Classification algorithms such as Logistic Regression, Decision Trees, and Random Forest can be applied for this task.

**Key Features:**

* Feature engineering using financial history data.
* Data preprocessing and exploratory data analysis.
* Model evaluation using metrics such as:

  * Precision
  * Recall
  * F1-Score
  * ROC-AUC
* Financial attributes considered include:

  * Income
  * Debts
  * Payment history
  * Other relevant financial indicators

**Repository:**
`CodeAlpha_AI-Credit-Scoring-System`

### 🔹 Task 2: Emotion Recognition from Speech

**Objective:**
Recognize human emotions (such as happy, angry, sad, and neutral) from speech audio.

**Approach:**
Applied deep learning and speech signal processing techniques to classify emotions from voice recordings.

**Key Features:**

* Audio preprocessing and feature extraction.
* Extraction of MFCCs (Mel-Frequency Cepstral Coefficients).
* Implementation of deep learning models such as:

  * CNN (Convolutional Neural Networks)
  * RNN (Recurrent Neural Networks)
  * LSTM (Long Short-Term Memory Networks)
* Utilized publicly available emotional speech datasets, including:

  * RAVDESS
  * TESS
  * EMO-DB

**Repository:**
`CodeAlpha_Emotion-Recognition-AI`
